<DESIGN_SPEC>
# Synchronous FIFO Design Specification

## 1. Module Name and Description
* **Module Name**: `sync_fifo`
* **Description**: A highly parameterized, production-grade Synchronous First-In, First-Out (FIFO) memory buffer. It features configurable data width, depth, and programmable status thresholds (`almost_full` and `almost_empty`). The design is split into a dedicated memory array module (`fifo_mem`) and a pointer/status control module (`fifo_ctrl`) to ensure clean physical synthesis and logical separation.

## 2. Port List

| Port Name | Direction | Width | Description |
| :--- | :--- | :--- | :--- |
| `clk` | Input | 1 | System clock (rising-edge active) |
| `rst_n` | Input | 1 | Asynchronous active-low reset |
| `w_en` | Input | 1 | Write enable |
| `w_data` | Input | `DATA_WIDTH` | Data input to be written |
| `r_en` | Input | 1 | Read enable |
| `r_data` | Output | `DATA_WIDTH` | Data output read from FIFO |
| `full` | Output | 1 | FIFO full flag (no more writes allowed) |
| `empty` | Output | 1 | FIFO empty flag (no more reads allowed) |
| `almost_full` | Output | 1 | Active when `data_count >= ALMOST_FULL_VAL` |
| `almost_empty`| Output | 1 | Active when `data_count <= ALMOST_EMPTY_VAL` |
| `data_count` | Output | `$clog2(FIFO_DEPTH) + 1` | Current number of elements in the FIFO |

## 3. Functional Description
* **Memory Architecture**: Uses a dual-port synchronous memory array (`fifo_mem`) with combinational read output (Standard FIFO behavior).
* **Pointer Logic**: Employs binary pointers with an extra MSB bit to distinguish between "Full" and "Empty" states when the read and write pointers point to the same address.
* **Status Flags**:
  * `empty` is asserted when the write pointer matches the read pointer exactly.
  * `full` is asserted when the write pointer and read pointer point to the same address but have different wrap-around MSBs.
  * `almost_full` and `almost_empty` are dynamically computed based on the current `data_count`.
* **Overflow/Underflow Protection**: Internal logic prevents write pointer increments when `full` is asserted, and read pointer increments when `empty` is asserted.

## 5. File Breakdown
1. `fifo_mem.sv`: Parameterized dual-port RAM block.
2. `fifo_ctrl.sv`: Pointer generation, status flag computation, and protection logic.
3. `sync_fifo.sv`: Top-level wrapper instantiating `fifo_mem` and `fifo_ctrl`.

## 6. Key Design Decisions
* **Power-of-2 Depth**: Enforced to allow clean pointer wrap-around logic using standard binary counters.
* **Modular Separation**: Separating memory from control logic allows ASIC/FPGA synthesis tools to easily map `fifo_mem` to dedicated Block RAMs (BRAM) or UltraRAMs.
* **Asynchronous Reset**: Active-low asynchronous reset is used to align with industry-standard cell libraries (Intel, TSMC, GlobalFoundries).

`timescale 1ns/1ps
/**
 * Module: fifo_mem
 * Description: Parameterized dual-port synchronous write, asynchronous read memory block.
 */
module fifo_mem #(
    parameter int DATA_WIDTH = 8,
    parameter int ADDR_WIDTH = 4
) (
    input  logic                  clk,
    input  logic                  w_en,
    input  logic [ADDR_WIDTH-1:0] w_addr,
    input  logic [DATA_WIDTH-1:0] w_data,
    input  logic [ADDR_WIDTH-1:0] r_addr,
    output logic [DATA_WIDTH-1:0] r_data
);

    localparam int DEPTH = 1 << ADDR_WIDTH;
    
    // Memory array declaration
    logic [DATA_WIDTH-1:0] mem [DEPTH-1:0];

    // Synchronous write port
    always_ff @(posedge clk) begin
        if (w_en) begin
            mem[w_addr] <= w_data;
        end
    end

    // Asynchronous read port (Standard FIFO behavior)
    assign r_data = mem[r_addr];

endmodule
</RTL_FILE>

`timescale 1ns/1ps
/**
 * Module: fifo_ctrl
 * Description: Pointer generation, status flag computation, and protection logic.
 */
module fifo_ctrl #(
    parameter int FIFO_DEPTH = 16,
    parameter int ALMOST_FULL_VAL = 12,
    parameter int ALMOST_EMPTY_VAL = 4,
    localparam int ADDR_WIDTH = $clog2(FIFO_DEPTH)
) (
    input  logic                  clk,
    input  logic                  rst_n,
    input  logic                  w_en,
    input  logic                  r_en,
    output logic [ADDR_WIDTH-1:0] w_addr,
    output logic [ADDR_WIDTH-1:0] r_addr,
    output logic                  full,
    output logic                  empty,
    output logic                  almost_full,
    output logic                  almost_empty,
    output logic [ADDR_WIDTH:0]   data_count
);

    // Pointers with an extra bit for wrap-around detection
    logic [ADDR_WIDTH:0] w_ptr;
    logic [ADDR_WIDTH:0] r_ptr;

    // Extract memory addresses from pointers
    assign w_addr = w_ptr[ADDR_WIDTH-1:0];
    assign r_addr = r_ptr[ADDR_WIDTH-1:0];

    // Write pointer logic
    always_ff @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            w_ptr <= '0;
        end else if (w_en && !full) begin
            w_ptr <= w_ptr + 1'b1;
        end
    end

    // Read pointer logic
    always_ff @(posedge clk or negedge rst_n) begin
        if (!rst_n) begin
            r_ptr <= '0;
        end else if (r_en && !empty) begin
            r_ptr <= r_ptr + 1'b1;
        end
    end

    // Status flags
    assign empty = (w_ptr == r_ptr);
    assign full  = (w_ptr[ADDR_WIDTH] != r_ptr[ADDR_WIDTH]) && 
                   (w_ptr[ADDR_WIDTH-1:0] == r_ptr[ADDR_WIDTH-1:0]);

    // Data count calculation (handles wrap-around automatically via 2's complement)
    assign data_count = w_ptr - r_ptr;

    // Programmable threshold flags
    assign almost_full  = (data_count >= ALMOST_FULL_VAL);
    assign almost_empty = (data_count <= ALMOST_EMPTY_VAL);

endmodule
</RTL_FILE>

/**
 * Module: sync_fifo
 * Description: Top-level wrapper for the Synchronous FIFO.
 */
module sync_fifo #(
    parameter int DATA_WIDTH = 8,
    parameter int FIFO_DEPTH = 16,
    parameter int ALMOST_FULL_VAL = 12,
    parameter int ALMOST_EMPTY_VAL = 4
) (
    input  logic                        clk,
    input  logic                        rst_n,
    input  logic                        w_en,
    input  logic [DATA_WIDTH-1:0]       w_data,
    input  logic                        r_en,
    output logic [DATA_WIDTH-1:0]       r_data,
    output logic                        full,
    output logic                        empty,
    output logic                        almost_full,
    output logic                        almost_empty,
    output logic [$clog2(FIFO_DEPTH):0] data_count
);

    localparam int ADDR_WIDTH = $clog2(FIFO_DEPTH);

    // Internal interconnects
    logic [ADDR_WIDTH-1:0] w_addr;
    logic [ADDR_WIDTH-1:0] r_addr;
    logic                  mem_w_en;

    // Write only if enabled and not full
    assign mem_w_en = w_en && !full;

    // Memory Instance
    fifo_mem #(
        .DATA_WIDTH(DATA_WIDTH),
        .ADDR_WIDTH(ADDR_WIDTH)
    ) u_fifo_mem (
        .clk    (clk),
        .w_en   (mem_w_en),
        .w_addr (w_addr),
        .w_data (w_data),
        .r_addr (r_addr),
        .r_data (r_data)
    );

    // Controller Instance
    fifo_ctrl #(
        .FIFO_DEPTH       (FIFO_DEPTH),
        .ALMOST_FULL_VAL  (ALMOST_FULL_VAL),
        .ALMOST_EMPTY_VAL (ALMOST_EMPTY_VAL)
    ) u_fifo_ctrl (
        .clk          (clk),
        .rst_n        (rst_n),
        .w_en         (w_en),
        .r_en         (r_en),
        .w_addr       (w_addr),
        .r_addr       (r_addr),
        .full         (full),
        .empty        (empty),
        .almost_full  (almost_full),
        .almost_empty (almost_empty),
        .data_count   (data_count)
    );

endmodule

module sync_fifo_tb;

    // Parameters
    localparam int DATA_WIDTH = 8;
    localparam int FIFO_DEPTH = 16;
    localparam int ALMOST_FULL_VAL = 12;
    localparam int ALMOST_EMPTY_VAL = 4;

    logic                  w_en;
    logic [DATA_WIDTH-1:0] w_data;
    logic                  r_en;
    logic [DATA_WIDTH-1:0] r_data;

    logic                  almost_empty;

    // Instantiate DUT
    sync_fifo #(
        .DATA_WIDTH(DATA_WIDTH),
        .FIFO_DEPTH(FIFO_DEPTH),
        .ALMOST_FULL_VAL(ALMOST_FULL_VAL),
        .ALMOST_EMPTY_VAL(ALMOST_EMPTY_VAL)
    ) dut (
        .clk(clk),
        .rst_n(rst_n),
        .w_en(w_en),
        .w_data(w_data),
        .r_en(r_en),
        .r_data(r_data),
        .full(full),
        .empty(empty),
        .almost_full(almost_full),
        .almost_empty(almost_empty),
        .data_count(data_count)
    );

    // Clock Generation
    always #5 clk = ~clk;

    // Watchdog Timer
    initial begin
        #100000;
        $display("ERROR: Watchdog timeout - simulation did not finish in time");
        $finish;
    end

    // Test Procedure
    initial begin
        // VCD Waveform Dump
        $dumpfile("sim/waves.vcd");
        $dumpvars(0, sync_fifo_tb);

        // Initialize signals
        clk = 0;
        rst_n = 1;
        w_en = 0;
        w_data = 0;
        r_en = 0;

        //---------------------------------------------------------
        // TEST CASE 1: Reset Verification
        //---------------------------------------------------------
        $display("[TC1] Starting Reset Verification...");
        rst_n = 0;
        #50;
        rst_n = 1;
        @(posedge clk);
        #1;
        if (!empty || full || (data_count != 0) || !almost_empty || almost_full) begin
            $display("ERROR: Reset state mismatch! empty=%b, full=%b, count=%d", empty, full, data_count);
            $finish;
        end
        $display("[TC1] Reset Verification Passed.");

        //---------------------------------------------------------
        // TEST CASE 2: Single Write and Single Read
        //---------------------------------------------------------
        $display("[TC2] Starting Single Write and Read...");
        @(posedge clk);
        #1;
        w_data = 8'hA5;
        w_en = 1;
        @(posedge clk);
        #1;
        w_en = 0;
        
        // Verify data count and flags
        if (data_count != 1 || empty) begin
            $display("ERROR: TC2 Write failed. count=%d, empty=%b", data_count, empty);
            $finish;
        end

        r_en = 1;
        @(posedge clk);
        #1;
        r_en = 0;
        if (r_data != 8'hA5 || !empty) begin
            $display("ERROR: TC2 Read failed. r_data=%h, empty=%b", r_data, empty);
            $finish;
        end
        $display("[TC2] Single Write and Read Passed.");

        //---------------------------------------------------------
        // TEST CASE 3: Fill to Full & Overflow Protection
        //---------------------------------------------------------
        $display("[TC3] Starting Fill to Full & Overflow Protection...");
        for (int i = 0; i < FIFO_DEPTH; i++) begin
            @(posedge clk);
            #1;
            w_data = i;
            w_en = 1;
        end
        @(posedge clk);
        #1;
        w_en = 0;

        if (!full || (data_count != FIFO_DEPTH) || !almost_full) begin
            $display("ERROR: FIFO not full! full=%b, count=%d", full, data_count);
            $finish;
        end

        // Attempt overflow write
        @(posedge clk);
        #1;
        w_data = 8'hFF;
        w_en = 1;
        @(posedge clk);
        #1;
        w_en = 0;

        if (!full || (data_count != FIFO_DEPTH)) begin
            $display("ERROR: Overflow corrupted FIFO state! full=%b, count=%d", full, data_count);
            $finish;
        end
        $display("[TC3] Fill to Full Passed.");

        //---------------------------------------------------------
        // TEST CASE 4: Empty to Empty & Underflow Protection
        //---------------------------------------------------------
        $display("[TC4] Starting Empty to Empty & Underflow Protection...");
        for (int i = 0; i < FIFO_DEPTH; i++) begin
            @(posedge clk);
            #1;
            r_en = 1;
            if (r_data != i) begin
                $display("ERROR: Data mismatch at index %d! Expected %d, Got %d", i, i, r_data);
                $finish;
            end
        end
        @(posedge clk);
        #1;
        r_en = 0;

        if (!empty || (data_count != 0) || !almost_empty) begin
            $display("ERROR: FIFO not empty! empty=%b, count=%d", empty, data_count);
            $finish;
        end

        // Attempt underflow read
        @(posedge clk);
        #1;
        r_en = 1;
        @(posedge clk);
        #1;
        r_en = 0;

        if (!empty || (data_count != 0)) begin
            $display("ERROR: Underflow corrupted FIFO state! empty=%b, count=%d", empty, data_count);
            $finish;
        end
        $display("[TC4] Empty to Empty Passed.");

        //---------------------------------------------------------
        // TEST CASE 5: Simultaneous Read and Write (Throughput)
        //---------------------------------------------------------
        $display("[TC5] Starting Simultaneous Read and Write...");
        // Pre-fill with 4 elements
        for (int i = 0; i < 4; i++) begin
            @(posedge clk);
            #1;
            w_data = 10 + i;
            w_en = 1;
        end
        @(posedge clk);
        #1;
        w_en = 0;

        // Simultaneous R/W for 10 cycles
        for (int i = 0; i < 10; i++) begin
            @(posedge clk);
            #1;
            w_data = 20 + i;
            w_en = 1;
            r_en = 1;
        end
        @(posedge clk);
        #1;
        w_en = 0;
        r_en = 0;

        if (data_count != 4) begin
            $display("ERROR: Simultaneous R/W count mismatch! Expected 4, Got %d", data_count);
            $finish;
        end
        $display("[TC5] Simultaneous Read and Write Passed.");

        $display("SUCCESS: All FIFO test cases passed successfully!");
        $finish;
    end

endmodule

`timescale 1ns/1ps
/**
 * Module: sync_fifo_sva
 * Description: SystemVerilog Assertions for sync_fifo to verify protocol compliance.
 */
module sync_fifo_sva #(
    parameter int DATA_WIDTH = 8,
    parameter int FIFO_DEPTH = 16,
    parameter int ALMOST_FULL_VAL = 12,
    parameter int ALMOST_EMPTY_VAL = 4
) (
    input  logic                        clk,
    input  logic                        rst_n,
    input  logic                        w_en,
    input  logic [DATA_WIDTH-1:0]       w_data,
    input  logic                        r_en,
    input  logic [DATA_WIDTH-1:0]       r_data,
    input  logic                        full,
    input  logic                        empty,
    input  logic                        almost_full,
    input  logic                        almost_empty,
    input  logic [$clog2(FIFO_DEPTH):0] data_count
);

    //---------------------------------------------------------
    // 1. Reset Property
    //---------------------------------------------------------
    property p_reset_state;
        @(posedge clk) !rst_n |-> (empty && !full && (data_count == 0) && almost_empty && !almost_full);
    endproperty
    assert_reset_state: assert property (p_reset_state)
        else $error("SVA ERROR: Reset state violation!");

    //---------------------------------------------------------
    // 2. Overflow Protection Property
    //---------------------------------------------------------
    property p_no_write_on_full;
        @(posedge clk) disable iff (!rst_n)
        (full && w_en && !r_en) |=> $stable(data_count);
    endproperty
    assert_no_write_on_full: assert property (p_no_write_on_full)
        else $error("SVA ERROR: Data count changed during write to full FIFO!");

    //---------------------------------------------------------
    // 3. Underflow Protection Property
    //---------------------------------------------------------
    property p_no_read_on_empty;
        @(posedge clk) disable iff (!rst_n)
        (empty && r_en && !w_en) |=> $stable(data_count);
    endproperty
    assert_no_read_on_empty: assert property (p_no_read_on_empty)
        else $error("SVA ERROR: Data count changed during read from empty FIFO!");

    //---------------------------------------------------------
    // 4. Full Flag Integrity
    //---------------------------------------------------------
    property p_full_flag;
        @(posedge clk) disable iff (!rst_n)
        (data_count == FIFO_DEPTH) |-> full;
    endproperty
    assert_full_flag: assert property (p_full_flag)
        else $error("SVA ERROR: Full flag not set when data_count equals FIFO_DEPTH!");

    //---------------------------------------------------------
    // 5. Empty Flag Integrity
    //---------------------------------------------------------
    property p_empty_flag;
        @(posedge clk) disable iff (!rst_n)
        (data_count == 0) |-> empty;
    endproperty
    assert_empty_flag: assert property (p_empty_flag)
        else $error("SVA ERROR: Empty flag not set when data_count is 0!");

    //---------------------------------------------------------
    // 6. Almost Full Flag Integrity
    //---------------------------------------------------------
    property p_almost_full_flag;
        @(posedge clk) disable iff (!rst_n)
        (data_count >= ALMOST_FULL_VAL) |-> almost_full;
    endproperty
    assert_almost_full_flag: assert property (p_almost_full_flag)
        else $error("SVA ERROR: Almost Full flag mismatch!");

    //---------------------------------------------------------
    // 7. Almost Empty Flag Integrity
    //---------------------------------------------------------
    property p_almost_empty_flag;
        @(posedge clk) disable iff (!rst_n)
        (data_count <= ALMOST_EMPTY_VAL) |-> almost_empty;
    endproperty
    assert_almost_empty_flag: assert property (p_almost_empty_flag)
        else $error("SVA ERROR: Almost Empty flag mismatch!");

    //---------------------------------------------------------
    // Cover Properties for Verification Completeness
    //---------------------------------------------------------
    cover_fifo_full: cover property (@(posedge clk) disable iff (!rst_n) full);
    cover_fifo_empty: cover property (@(posedge clk) disable iff (!rst_n) empty);
    cover_simultaneous_rw: cover property (@(posedge clk) disable iff (!rst_n) (w_en && r_en && !full && !empty));

endmodule

// Bind SVA to the top-level design

    .ALMOST_FULL_VAL(ALMOST_FULL_VAL),
    .ALMOST_EMPTY_VAL(ALMOST_EMPTY_VAL)
) u_sync_fifo_sva (
    .clk(clk),
    .rst_n(rst_n),
    .w_en(w_en),
    .w_data(w_data),
    .r_en(r_en),
    .r_data(r_data),
    .full(full),
    .empty(empty),
    .almost_full(almost_full),
    .almost_empty(almost_empty),
    .data_count(data_count)
);

# Functional Coverage Plan - Synchronous FIFO

## 1. Coverage Points Table

| Coverage Point | Description | Bin Definitions / Target |
| :--- | :--- | :--- |
| `cp_write` | Write Enable activity | `bin write_active = {1}` |
| `cp_read` | Read Enable activity | `bin read_active = {1}` |
| `cp_data_count` | Occupancy level of the FIFO | `bins empty = {0}`, `bins partial = {[1:15]}`, `bins full = {16}` |
| `cp_almost_full` | Almost Full flag transitions | `bins active = {1}`, `bins inactive = {0}` |
| `cp_almost_empty`| Almost Empty flag transitions | `bins active = {1}`, `bins inactive = {0}` |
| `cross_rw` | Cross coverage of R/W operations | Combinations of (Read, Write) when FIFO is empty, partial, or full |

## 2. SystemVerilog Covergroup Definition

```systemverilog
covergroup fifo_cov_grp @(posedge clk);
    option.per_instance = 1;
    option.goal = 100;

    // Coverpoint for Write Enable
    cp_w_en: coverpoint w_en {
        bins active = {1'b1};
        bins inactive = {1'b0};
    }

    // Coverpoint for Read Enable
    cp_r_en: coverpoint r_en {
        bins active = {1'b1};
        bins inactive = {1'b0};
    }

    // Coverpoint for FIFO occupancy
    cp_occupancy: coverpoint data_count {
        bins empty = {0};
        bins almost_empty_zone = {[1:ALMOST_EMPTY_VAL]};
        bins mid_zone = {[ALMOST_EMPTY_VAL+1:ALMOST_FULL_VAL-1]};
        bins almost_full_zone = {[ALMOST_FULL_VAL:FIFO_DEPTH-1]};
        bins full = {FIFO_DEPTH};
    }

    // Cross coverage to ensure simultaneous reads and writes are verified across occupancy levels
    cross_rw_occupancy: cross cp_w_en, cp_r_en, cp_occupancy {
        ignore_bins empty_read = binsof(cp_occupancy.empty) && binsof(cp_r_en.active) && binsof(cp_w_en.inactive);
        ignore_bins full_write = binsof(cp_occupancy.full) && binsof(cp_w_en.active) && binsof(cp_r_en.inactive);
    }
endgroup
```

## 3. Corner Cases to Cover
* **Simultaneous Read and Write on Empty**: Verifies that the FIFO handles simultaneous R/W when empty without underflowing or dropping data.
* **Simultaneous Read and Write on Full**: Verifies that the FIFO handles simultaneous R/W when full without overflowing or blocking the write.
* **Back-to-back Writes to Full**: Verifies that the overflow protection logic blocks subsequent writes once `full` is asserted.
* **Back-to-back Reads to Empty**: Verifies that the underflow protection logic blocks subsequent reads once `empty` is asserted.
</COVERAGE_PLAN>

# Synchronous FIFO Project

This project implements a highly parameterized, production-ready Synchronous FIFO in SystemVerilog. The design is modularized into separate files for memory and control logic, and includes a comprehensive self-checking testbench, SystemVerilog Assertions (SVA) for formal/dynamic protocol checking, and a functional coverage plan.

## Directory Structure
```text
├── fifo_mem.sv        # Parameterized dual-port RAM block
├── fifo_ctrl.sv       # Pointer and status flag generation logic
├── sync_fifo.sv       # Top-level FIFO wrapper
├── sync_fifo_tb.sv    # Self-checking testbench with 5 test cases
├── sync_fifo_sva.sv   # SystemVerilog Assertions (bound to top-level)
└── README.md          # Project documentation
```

## How to Run Simulation

You can compile and run the simulation using any standard IEEE SystemVerilog 2012 compliant simulator (e.g., Icarus Verilog, ModelSim, VCS, Questa).

### Running with Icarus Verilog (v10.0+)

1. Create a simulation directory:
   ```bash
   mkdir -p sim
   ```

2. Compile the design, assertions, and testbench files:
   ```bash
   iverilog -g2012 -o sim/fifo_sim fifo_mem.sv fifo_ctrl.sv sync_fifo.sv sync_fifo_sva.sv sync_fifo_tb.sv
   ```

3. Run the simulation:
   ```bash
   vvp sim/fifo_sim
   ```

4. View waveforms using GTKWave:
   ```bash
   gtkwave sim/waves.vcd
   ```
